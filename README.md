## 🛠️ Investigation Framework

For every challenge, I adhere to a structured Incident Response workflow:

1.  **Alert Triage**: Initial assessment of the alert to determine its scope and impact.
2.  **Evidence Collection**: Gathering artifacts from SIEM, EDR, or Firewall logs.
3.  **Deep Analysis**: Correlating logs, performing static/dynamic analysis, and identifying malicious patterns.
4.  **Reporting**: Providing clear answers to challenge questions and suggesting remediation steps.

---

## 📝 Writeup Structure

Each investigation folder in this repository follows a consistent format:

* **Executive Summary**: A high-level overview of the incident.
* **Artifacts**: A list of discovered IOCs (IPs, Domains, Hashes).
* **Methodology**: Step-by-step technical breakdown of the investigation.
* **Q&A**: Solutions to the specific challenge questions.
